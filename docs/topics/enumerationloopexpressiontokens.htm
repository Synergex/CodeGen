Enumeration Loop Expression Tokens

 

Enumeration loop expressions are expressions that can only be used within the context of an enumeration loop and allow you to conditionally include or exclude code based on the attributes of the current enumeration.

Enumeration Loop Expression

Description

<IF DESCRIPTION>

The enumeration has descriptive text specified.

<IF LONG_DESCRIPTION>

The enumeration has a long description specified.
